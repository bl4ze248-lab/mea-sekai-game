# Mea Sekai Game

A 2D adventure game built with Phaser 3 and JavaScript.

## 🎮 About the Game

Mea Sekai is an immersive 2D game experience where players explore a mysterious world, solve puzzles, and uncover secrets.

## 🚀 Quick Start

### Prerequisites
- Node.js (v14 or higher)
- npm

### Installation

```bash
git clone https://github.com/bl4ze248-lab/mea-sekai-game.git
cd mea-sekai-game
npm install
```

### Running the Game

```bash
npm start
```

The game will open at `http://localhost:8000`

## 📁 Project Structure

```
mea-sekai-game/
├── index.html          # Main HTML entry point
├── main.js            # Game initialization
├── package.json       # Dependencies
├── assets/            # Game assets
│   ├── images/        # Sprites and textures
│   ├── audio/         # Music and sound effects
│   └── tilemaps/      # Level data
└── src/               # Game source code
    ├── scenes/        # Game scenes
    ├── entities/      # Game objects
    └── utils/         # Helper functions
```

## 🛠️ Development

### Adding New Scenes
Create new scene files in `src/scenes/` and import them in `main.js`.

### Adding Assets
Place your images, audio, and tilemap files in the `assets/` folder.

### Building for Production
```bash
npm run build
```
